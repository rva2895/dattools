Collection of utilities for Age of Empires II/Star Wars Galactic Battlegrounds SLP and DAT files

slpshrink: optimise SLP images
terrainmap: create terrain SLP from bitmap image
colorreplace: replace colors in SLP images
icmtool: create VIEW_ICM.DAT from palette
shadowcol: create SHADOW.COL from palette
makebln: create BLN file from palette
